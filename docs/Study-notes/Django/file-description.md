# 项目文件夹/文件说明

# ![django-project](img\django-project.png)

**demo02:**项目文件夹<br>
**app01/app02:**应用文件夹<br>
**migrations:**数据库迁移记录文件夹<br>
**admin.py:**后台管理<br>
**apps.py:**注册应用<br>
**models.py:**数据库相关的模型类<br>
**tests.py:**测试文件<br>
**views.py:**视图函数<br>
**static:**静态数据存放文件夹<br>
**templates:**html文件存放路径<br>
**setting.py:**配置文件<br>
**url.py:**路由和视图的对应关系<br>
**wsgi.py:**wsgire模块<br>
